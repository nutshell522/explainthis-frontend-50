# ExplainThis 50 題前端程式手寫題

此專案來源於 [ExplainThis 50 題前端程式手寫題](https://explainthisio.notion.site/ExplainThis-50-8fe7055e22d5467586f7d2c22719684f)，主要目標是練習與理解前端 JavaScript 的基礎與進階概念。

## 專案簡介

本專案包含 50 題前端手寫題，並提供詳細的解法與解析，涵蓋以下主題：

- **Part 1 - Lodash 手寫題**
- **Part 2 - 陣列 (Array)**
- **Part 3 - Promise 與 Time**
- **Part 4 - 函式 (Function)**
- **Part 5 - 閉包 (Closure)**
- **Part 6 - 物件與類別 (Object and Class)**
